# 魔兽编辑器语法高亮 WeSyntax
SublimeText 的 vJass 语法高亮，使用 YAML 制作。  
默认会对 \*.j 文件使用 vJass 的高亮，你也可以点击 Sublime 的右下角来手动选择语法。

## 使用方法
把 vjass 和 weText 这两个文件夹，拷贝到文件夹 %USERPROFILE%\AppData\Roaming\Sublime Text 3\Packages 里面。
